## 摘要
模型内部状态编码了关于大语言模型（LLM）如何处理其训练数据的丰富信息；然而，后训练数据工程主要依赖外部信号，而忽略了模型内部蕴含的丰富内在信号。我们提出SAERL，一个用于LLM强化学习（RL）的数据工程框架。它利用稀疏自编码器（SAE）这一先进的机制可解释性工具提取模型内部状态，从而建模三种内在数据属性：多样性、难度和质量。每种属性支撑一项具体的数据工程操作：通过适度批混合的SAE空间聚类实现批次多样性控制，作为从易到难课程排序的难度代理，以及用于数据过滤的质量探针。SAERL在Qwen2.5-Math-1.5B上相比原始GRPO平均准确率提升3.00%，并以减少20%训练步骤达到目标准确率，且在不同模型规模和RL算法上均有一致提升。实验表明，SAE在跨模型族和规模时能有效迁移，成为一种轻量级且可复用的数据工程工具。这些结果证明，模型内部状态是后训练数据工程中强大且实用的信号来源。

## Abstract
Model internals encode rich information about how a large language model (LLM) processes its training data; however, post-training data engineering largely relies on external signals and ignores rich intrinsic signals lying in model internals. We propose SAERL, a data engineering framework for LLM reinforcement learning (RL). It models three intrinsic data properties: diversity, difficulty, and quality, using model internals extracted with Sparse Autoencoder (SAE), an advanced mechanistic interpretability tool. Each property grounds a concrete data engineering operation: SAE-space clustering with moderate batch mixing for batch diversity control, a difficulty proxy for easy-to-hard curriculum ordering, and a quality probe for data filtering. SAERL improves average accuracy by 3.00% over vanilla GRPO and reaches target accuracy with 20% fewer training steps on Qwen2.5-Math-1.5B, with consistent gains across model scales and RL algorithms. Experiments show that SAE transfers effectively across model families and scales, serving as a lightweight and reusable data engineering tool. These results demonstrate that model internals are a powerful and practical source of signals for post-training data engineering.

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：使用SAE将LLM在特定层的隐藏表征分解为稀疏的、细粒度的特征激活。对每个数据样本，提取其prompt和solution两侧的SAE激活，并通过均值池化和最大池化聚合，再拼接少量元数据（如长度、TeX比例等），形成统一表征 `ϕ(x_i)`。基于该表征建模三种内在数据属性，分别对应三项数据工程操作：
  1. **多样性**：在SAE空间中对样本进行聚类（MiniBatchKMeans，K=10），然后构建簇优先的批次，并引入**适度批次混合**（交换批次尾部的一小部分样本与相邻批次），以平衡批次内的梯度一致性和跨簇覆盖。
  2. **难度**：在一个小型标注子集（|L|=3000）上训练一个**ElasticNet回归器**，预测每个样本的原始难度分数；再通过全局校准和基于簇的收缩残差调整，得到最终排序分数 `r_i`，用于簇内从易到难的课程排序。

1. **模型内部信号（SAE激活）可有效用于后训练数据工程**：建模多样性、难度、质量并分别指导批次构建、课程排序和数据过滤，显著提升准确率和训练效率。

## 8. 不足与局限

- **领域局限**：实验仅限数学推理（可验证奖励），未在代码RL、智能体RL、工具使用、通用指令遵循等场景验证。SAE特征在不同领域中是否仍能有效表征多样性、难度和质量尚不清楚。
- **非完全无监督**：难度代理仍需少量人工标签（3000样本），质量探针需要源/分布标签。虽比推出式方法成本低，但仍依赖一定监督信号。未来可探索无监督或自校准方法。
- **理论分析有限**：对批次多样性的解释基于假设（SAE空间近似梯度相干性），并未直接测量梯度，缺少因果证明。文中虽给出了偏置-方差分解（附录A），但仍是理论推导而非实证验证。
- **SAE训练依赖**：SAE训练需要特定语料和模型层，如果目标模型结构差异很大（如MoE、不同词汇表），迁移效果可能下降。论文仅使用了Qwen3-1.7B训练SAE并迁移到Qwen2.5-Math，未测试更大差异的情况。
- **实验覆盖**：未与更多基于梯度的数据选择方法（如LESS、GradAlign）对比；批次大小分析仅做了128和512，未测试更极端值；未报告训练的总GPU消耗或时间（仅给出前置成本）。
- **偏差风险**：SAE特征可能存在对某些类型数据的偏好（如长回答或特定符号结构），导致过滤或排序引入系统性偏差。
