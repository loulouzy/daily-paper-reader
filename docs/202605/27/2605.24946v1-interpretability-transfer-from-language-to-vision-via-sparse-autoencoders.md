## 摘要
近年来，使用稀疏自编码器（SAE）进行语言模型可解释性研究取得了进展，但这一方法尚未有效迁移到视觉领域，主要原因在于视觉概念标注的困难性和模糊性。本文提出了视觉可解释性通过SAE迁移对齐（VISTA）框架，通过在LLaVA风格的视觉-语言模型中约束视觉投影器，将视觉令牌映射到LLM已有的、带有标签的文本SAE空间中，从而无需训练专门的视觉SAE即可实现视觉可解释性。通过利用LLM的SAE重建损失对投影器进行正则化，VISTA将匹配率（衡量SAE空间中激活最强的文本概念与图像中语义元素对应准确度的指标）提升了三倍。借助该框架，我们进一步分析了不同视觉编码器的空间定位特性，结果表明DINOv2特征具有比其他编码器更强的定位能力。利用这种精确性，我们通过细粒度的局部概念干预验证了VISTA的跨模态对齐能力——在保留周围场景的同时，从模型感知中移除或替换特定对象。与纯视觉基线相比，对象移除任务性能提升35%，对象替换任务性能提升47%，为视觉令牌存在于文本SAE流形中提供了因果证据。这些贡献在多种LLM架构上得到了验证。

## Abstract
Recent advances in language model interpretability using sparse autoencoders (SAEs) have yet to effectively translate to the visual domain, mainly due to the difficulty and ambiguity of labeling visual concepts. In this paper, we introduce Visual Interpretability via SAE Transfer Alignment (VISTA), a framework that transfers interpretability from language to vision in a LLaVA-style vision-language model by constraining a visual projector to map visual tokens into an LLM's pre-existing, labeled textual SAE space. This approach enables visual interpretability without training dedicated vision SAEs. By regularizing the projector using the LLM's SAE reconstruction loss, VISTA achieves a threefold increase in the matching rate, which measures how accurately the most activating textual concepts in the SAE space correspond to semantic elements in the image. Using this framework, we further analyze spatial localization properties of different vision encoders and show that DINOv2 features have stronger localization abilities than other encoders. Leveraging this precision, we validate VISTA's cross-modal alignment through fine-grained, localized concept interventions, where specific objects are removed or replaced in the model's perception while preserving the surrounding scene. This results in improvements of 35% in object removal and 47% in object replacement tasks over vision-only baselines, providing causal evidence that visual tokens inhabit the text SAE manifold. These contributions are validated across multiple LLM architectures.

- **研究动机**：大语言模型（LLM）的可解释性已通过稀疏自编码器（SAE）取得显著进展，但视觉领域因概念标注困难、模糊性高，难以直接训练专用视觉SAE。现有两条路径：一是训练视觉编码器的SAE（标注困难且无法揭示视觉特征进入LLM残差流后的解释），二是微调已有文本SAE融入视觉信息（计算成本极高，如Gemma Scope需约20% GPT-3算力及大量人工标注）。核心问题：**能否复用语言SAE来解释视觉概念？**
- **整体含义**：论文提出VISTA框架，通过约束视觉投影器将视觉token映射到LLM已有的、带有标签的文本SAE空间，实现“可解释性迁移”。该方法避免了训练或微调专用视觉SAE，显著降低了计算和标注成本，并在匹配率、空间定位、局部操控上取得了大幅提升，为多模态模型的可解释性与因果控制提供了可扩展路径。

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：在LLaVA风格的VLM中，保持视觉编码器和LLM冻结，仅训练一个线性投影器。VISTA在标准交叉熵损失（$L_{CE}$）基础上增加一个**SAE重构损失（$L_{SAE}$）**，强制视觉token的隐藏状态能够被预训练的文本SAE良好重构，从而对齐到文本SAE的稀疏语义流形。
- **关键技术细节**：
  - 架构：视觉编码器（V，如DINOv2）产生视觉token $v \in \mathbb{R}^{N\times D_v}$，经可学习线性投影器 $P_\theta$ 映射到LLM嵌入空间：$e_v = P_\theta(v) \in \mathbb{R}^{N\times D_{\text{llm}}}$。
  - SAE重构损失：$L_{SAE} = \frac{1}{|M|} \sum_{l \in M} \| \text{SAE}_l(e_v^l) - e_v^l \|_2^2$，其中 $l$ 为LLM层索引，$M$ 为施加约束的层集合（实验中选择前5层）。SAE为在文本上预训练的编码器-解码器结构。
  - 总目标：$L_{\text{total}} = L_{CE} + L_{SAE}$。
  - 训练两阶段：预训练使用595k CC3M图像-文本对；微调使用665k LLaVA-1.5样本。
  - 局部视觉操控：训练后，可直接在SAE空间中通过算术操作（加/减特定SAE解码器方向）修改视觉token，实现对象移除或替换，且仅影响选定的空间区域（patch），保持全局场景不变。

1. **可解释性迁移成功**：VISTA将匹配率（LLM验证的概念-图像对应率）平均提升三倍（如DINOv2+Gemma-2B从约0.2提升至约0.6以上）。

- **方法简洁**：仅增加一项辅助损失，无需修改模型架构，可即插即用于现有LLaVA流水线。

## 8. 不足与局限

- **依赖预训练文本SAE**：需要目标LLM已有公开SAE（如Gemma Scope、LLaMA Scope），限制了方法对未开源SAE的模型的应用。论文指出“随着基础模型与SAE配对发布，这一限制正减弱”。

- **对比方法局限**：仅与视觉SAE方法（VisionSAE、PatchSAE）对比，未与非SAE操控方法（如CAA、激活添加）或端到端微调方法比较。
