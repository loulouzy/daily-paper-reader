## 摘要
视觉结果越来越成为多模态大语言模型的核心，因此可靠且细粒度的验证对于扩展通用基础模型至关重要。在本工作中，我们研究了多模态元验证，即利用验证器生成的推理而非仅决策信号，并探索如何将元验证反馈有效融入多模态验证器训练。我们发现了两个关键结果。第一，符号验证器输出（如边界框）作为元验证推理优于文本解释，能够实现基于规则的强化学习奖励，同时避免依赖来自辅助评判模型的模型奖励。第二，将二元判断和元验证的强化学习目标解耦显著优于联合奖励优化，这是由于输出结构和学习动态的内在差异。基于这些见解，我们训练了OmniVerifier-M1，这是一个利用符号元验证和解耦强化学习的通用视觉验证器。OmniVerifier-M1提供稳健的验证和细粒度的错误定位，并进一步实现了M1-TTS，这是一个验证器驱动的代理生成系统，能够实现动态区域级自我纠正。该方法为更可靠、可解释和细粒度的多模态验证铺平了道路，支持更安全、更可控的基础模型部署。

## Abstract
Visual outcomes are increasingly central to multimodal large language models, making reliable and fine-grained verification essential for scaling generalist foundation models. In this work, we investigate multimodal meta-verification, which leverages verifier-generated rationales rather than decision-only signals, and explore how to effectively incorporate meta-verification feedback into multimodal verifier training. We identify two key findings. First, symbolic verifier outputs (e.g., bounding boxes) outperform textual explanations as meta-verification rationales, enabling efficient rule-based reinforcement learning rewards while avoiding reliance on model-based rewards from auxiliary judge models. Second, decoupling reinforcement learning objectives for binary judgment and meta-verification substantially outperforms joint reward optimization, due to intrinsic differences in output structure and learning dynamics. Based on these insights, we train OmniVerifier-M1, a generalist visual verifier leveraging symbolic meta-verification and decoupled reinforcement learning. OmniVerifier-M1 provides robust verification and fine-grained error localization, and further enables M1-TTS, a verifier-driven agentic generation system achieving dynamic region-level self-correction. This approach paves the way for more reliable, interpretable, and fine-grained multimodal verification, supporting safer and more controllable foundation model deployment.

- **核心问题**：多模态大语言模型（MLLMs）的视觉输出（如图像）需要可靠且细粒度的验证。现有的视觉验证方法多依赖二元判断（True/False）或仅输出标量奖励，缺乏可解释的细粒度推理，监督信号稀疏，容易导致模型学习到表面模式而非真正的验证能力。
- **研究动机**：引入**多模态元验证**（Multimodal Meta-Verification）概念，即利用验证器自己生成的推理（而不是仅决策信号）作为训练反馈，以提供更密集、更严格的监督，提升验证器的可靠性、可解释性和细粒度错误定位能力。
- **整体含义**：本文为多模态验证提供了一种新的训练范式，通过符号化输出和解耦强化学习，在提升验证精度的同时降低训练成本，并进一步构建了验证器驱动的细粒度图像纠正系统，为更安全可控的基础模型部署铺平道路。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：
  1. **符号化元验证推理**：使用边界框（bounding box）或点等符号输出作为验证失败区域的解释，替代文本解释。符号输出可直接通过规则（如IoU）计算奖励，避免依赖额外的模型评估器，消除奖励破解风险。
  2. **解耦强化学习训练**：将二元判断任务和元验证任务（生成符号定位）拆分为两个独立目标，各自使用独立的奖励信号进行训练，而非将两者耦合在同一个样本的同一个奖励中。

- **关键技术细节**：
  - **基础框架**：基于RLVR（Reinforcement Learning with Verifier Rewards）范式，使用DAPO算法训练视觉验证器。
  - **元验证奖励**：对于负样本（ground-truth False），验证器需输出边界框。规则奖励采用预测框与真实框的IoU，设置阈值0.6得到二元奖励（0/1）。
  - **解耦策略**：原始数据集正负样本各占一半。所有负样本被复制一份：一份用于二元判断任务（仅由准确率奖励监督），另一份专门用于元验证任务（仅由IoU奖励监督）。在训练中混合两个数据流。

## 6. 论文的主要结论与发现

1. **符号输出优于文本解释**：在多模态元验证中，使用边界框等符号输出作为推理，可获得与文本解释相当的验证精度，且规则奖励更高效、无奖励破解风险。
2. **解耦训练优于联合训练**：将二元判断和元验证作为独立任务解耦训练，在ViVerBench和RefCOCO上均显著优于联合训练。理论证明解耦提供了更高的梯度信噪比和更稳定的优化。
3. **OmniVerifier-M1**：基于Qwen3-VL-8B训练得到的验证器在ViVerBench上达到0.68（提升约3-4%），在RefCOCO上达到0.866（提升约4-5%）。
4. **M1-TTS**：验证器驱动的agentic生成系统，通过符号定位和结构化编辑指令实现区域级迭代纠正，在WISE和T2I-CoreBench上超越纯生成模型和传统全局编辑方法。
