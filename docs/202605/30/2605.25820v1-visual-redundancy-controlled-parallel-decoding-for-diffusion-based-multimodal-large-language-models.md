## 摘要
基于扩散的多模态大型语言模型（dMLLMs）通过迭代地在多个掩码位置并行预测令牌进行解码。这使每个解码步骤变为位置选择问题：模型不仅需要独立判断哪些预测是可靠的，还需要决定哪些位置应被共同提交，作为后续解码步骤的上下文。现有的基于置信度的解码方法独立地对掩码位置进行排序，并提交前K个位置，很大程度上忽略了所提交的令牌是否提供互补的视觉基础。我们发现在多模态设置中，这种策略存在步骤级限制：同一步骤中选择的高置信度令牌可能依赖重叠的视觉基础，导致提交的令牌之间存在视觉冗余，使得后续解码可用的互补视觉基础减少。为了量化这一影响，我们引入了视觉冗余指数（VRI），它衡量并行提交的令牌之间的视觉基础重叠程度。为了在解码过程中控制这种冗余，我们提出了视觉冗余控制解码（VRCD），这是一种无需训练、推理时的解码方法，利用令牌到图像的注意力来优先选择视觉互补的位置。在多种多模态基准测试中，VRCD以适度的运行时开销减少了视觉冗余和剩余位置的熵。在更长的解码实验中，与基于置信度的解码相比，它在M^3CoT上实现了高达18.8%的相对准确率提升，在MMBench上实现了6.9%的提升。代码将在https://github.com/infiniteYuanyl/VRCD发布。

## Abstract
Diffusion-based multimodal large language models (dMLLMs) decode by iteratively predicting tokens at multiple masked positions in parallel. This turns each decoding step into a position-selection problem: the model must choose not only which predictions are reliable in isolation, but also which positions should be committed together as context for later decoding steps. Existing confidence-based decoding ranks masked positions independently and commits the top-K positions, largely ignoring whether the committed tokens provide complementary visual grounding. We identify a step-level limitation of this strategy in multimodal settings: high-confidence tokens selected in the same step can rely on overlapping visual grounding, introducing visual redundancy among the committed tokens and leaving less complementary visual grounding available for later decoding. To quantify this effect, we introduce the Visual Redundancy Index (VRI), which measures visual grounding overlap among tokens committed in parallel. To control this redundancy during decoding, we propose Visual-Redundancy-Controlled Decoding (VRCD), a training-free inference-time decoding method that uses token-to-image attention to prioritize visually complementary positions. Across diverse multimodal benchmarks, VRCD reduces visual redundancy and remaining-position entropy with modest runtime overhead. In longer decoding experiments, it also achieves relative accuracy gains of up to 18.8% on M^3CoT and 6.9% on MMBench over confidence-based decoding. Code will be released at https://github.com/infiniteYuanyl/VRCD.

## 1. 核心问题与整体含义（研究动机和背景）
- **问题**：扩散式多模态大语言模型（dMLLMs）采用并行解码，在每个步骤同时提交多个位置的预测结果。现有基于置信度的解码方法独立排序每个掩码位置的置信度并提交top-K，忽略了这些位置之间视觉基础（visual grounding）的互补性。
- **现象**：多个高置信度token可能关注图像中相同的区域（如同一个物体），导致提交的token携带重叠的视觉信息，即“视觉冗余”。这会使后续解码步骤缺乏多样化的视觉上下文，限制解码质量。
- **研究意义**：首次系统性地识别并量化了dMLLM并行解码中的步骤级视觉冗余问题，并提出无训练推理时解码方法来缓解该问题，从而提升解码质量和效率。

## 2. 方法论：核心思想、关键技术细节与算法流程
### 核心思想
利用令牌到图像的交叉注意力（token-to-image attention）估计视觉重叠程度，通过置信度重加权优先选择视觉互补的位置提交。

### 关键技术细节
- **视觉冗余指数（VRI）**：度量并行提交的token之间视觉基础的重叠程度。对每个图像token，计算所有提交token的注意力权重之和减去最大单token贡献，再归一化到[0,1]。
- **VRCD解码流程**（无训练，推理时reranking）：
  1. **候选窗口构建**：从所有掩码位置中选出高置信度的前M_t个位置（M_t = min(|C_t|, max(K_t, ⌈λK_t⌉))），λ控制窗口大小。
  2. **视觉显著性提取**：对每个候选位置，从最终解码器层的token-to-image注意力（多头平均）中，减去均匀分量1/N，只保留正残差并重新归一化，得到视觉显著性分布q_i(u)。
  3. **两两视觉重叠计算**：使用Bhattacharyya系数：o_ij = Σ_u sqrt(q_i(u)*q_j(u))。
  4. **冗余分数计算**：将每个重叠值转换为当前窗口内的百分位排名R_t(i,j)；然后对每个候选i，聚合其与窗口内其他候选的加权重叠：r_i = Σ_{j∈B_i} (c_j / Z_i) * R_t(i,j)，其中c_j是置信度。该分数衡量候选i与其他可能被提交的高置信度候选的视觉重叠程度。
  5. **冗余控制分数**：s_i = c_i * (r_i + 1)^{-α}，α控制冗余惩罚强度。按s_i降序选择K_t个位置提交。

### 关键公式
- Eq. (2): VRI定义
- Eq. (3): 候选窗口大小
- Eq. (5): 视觉显著性提取
- Eq. (6): token级冗余分数r_i
- Eq. (7): 冗余控制分数s_i

### 算法流程（以文字描述）
每个解码步骤：输入掩码位置、预测令牌及其置信度 → 按置信度截取候选窗口 → 提取每个候选的视觉显著性 → 计算候选对视觉重叠 → 转换为百分位秩 → 计算每个候选的冗余分数 → 用α指数重加权置信度 → 选择胜出的K_t个位置提交。
