## 摘要
视觉-语言模型（VLM）日益驱动着从医学影像到自动驾驶系统等高风险应用，但它们经常产生幻觉，自信地描述输入中不存在的内容。我们通过聚焦于基于解码器的VLM的机制分析，探究了这些失败模式的根源。我们将这些失败模式追溯至几何上的过度对齐：为了弥合注意力机制所需的模态间隙，基于解码器的VLM将视觉嵌入与文本流形过度对齐，从而注入了一种统计上的语言偏差，系统性地掩盖了细粒度的视觉证据。尽管先前的工作要么激进地缩小这一间隙，要么通过昂贵的黑盒解码策略抑制幻觉，但均未解决底层的几何原因。我们首次对此过度对齐进行了定量刻画，证明了语言偏差集中在一个通用的、与数据集无关的文本子空间的顶部主成分中。基于这一洞见，我们提出了两种互补的补救措施：一种免训练推理策略和一种感知偏差的微调范式，两者均明确地将该子空间从视觉表示中投影出去。我们的方法在POPE、CHAIR和AMBER基准上显著减少了幻觉，并提高了长描述任务中的CLAIR分数，其中免训练变体在基础模型上不增加计算开销。

## Abstract
Vision-Language Models (VLMs) increasingly power high-stakes applications, from medical imaging to autonomous systems, yet they routinely hallucinate, confidently describing content not present in the input. We investigate the root causes of these failure modes with a mechanistic analysis focusing on the decoder-based VLMs. We trace these failure modes to a geometric over-alignment: to bridge the modality gap required by attention mechanisms, decoder-based VLMs over-align visual embeddings with the text manifold, injecting a statistical linguistic bias that systematically overshadows fine-grained visual evidence. While prior work either aggressively closes this gap or suppresses hallucinations through expensive black-box decoding strategies, none addresses the underlying geometric cause. We provide the first quantitative characterization of this over-alignment, demonstrating that linguistic bias concentrates in the top principal components of a universal, dataset-agnostic text subspace. Building on this insight, we propose two complementary remedies: a training-free inference strategy and a bias-aware fine-tuning paradigm, both of which explicitly project out this subspace from visual representations. Our methods significantly reduce hallucinations across POPE, CHAIR, and AMBER benchmarks, and improve CLAIR scores on long-form captioning tasks, with the training-free variant adding no computational overhead over the base model.

- **整体含义**：本文首次定量刻画了这一几何过度对齐现象，并提出了针对性的几何去偏框架，为理解 VLM 幻觉提供了新视角。

## 2. 方法论
### 2.1 核心思想
- 假设语言偏差是线性可分离的，基于**线性表示假设**，通过主成分分析（PCA）从预训练文本中提取一个通用的、与数据集无关的文本子空间。顶部主成分承载了统计性文本先验，而低阶主成分则包含语义信息。
- 通过将视觉嵌入向该子空间的顶部主成分进行正交投影（即减掉投影分量），可以去除语言偏差，使视觉表示恢复对真实视觉特征的编码。

### 2.2 关键技术细节
1. **文本流形定义**：从预训练数据集的图像描述中提取文本嵌入，对其执行 PCA，得到前 K 个主成分 \[ \mathbf{k}_1,\dots,\mathbf{k}_K \]。
2. **对齐度量**：定义对齐分数 \(\text{Align}^{(l)} = \frac{1}{M}\sum_{j} \frac{\|\mathbf{z}_j^{(l)}\|_2}{\|\mathbf{v}_j^{(l)} - \mathbf{m}\|_2}\)，其中 \(\mathbf{z}^{(l)}\) 是视觉嵌入 \(\mathbf{v}^{(l)}\) 在文本流形上的投影，\(\mathbf{m}\) 是文本均值。高对齐分数表示视觉嵌入被强行拉入文本空间。
3. **训练‑免推理去偏**：对每个视觉嵌入 \(\mathbf{v}\)，计算 \[ \mathbf{v}_{\text{debiased}} = \mathbf{v} - \sum_{i=1}^k \big((\mathbf{v} - \mathbf{m}) \cdot \mathbf{k}_i\big) \mathbf{k}_i \]，减去前 k 个主成分上的投影分量，无需额外训练，不增加推理开销。
4. **偏差感知微调**：在微调时，对投影器的输出在送入 LLM 解码器之前执行上述去偏操作，剥夺网络使用文本统计捷径的能力，迫使投影器学习纯粹视觉语义。

### 2.3 算法流程（文字说明）
- **训练‑免方法**：加载预训练 VLM → 从训练集描述提取文本 → PCA 计算前 k 个主成分 → 推理时，在 LLM 每一层或仅在特定层对视觉隐藏状态进行中心化并减去顶部主成分投影 → 得到去偏后的表示继续生成。
- **微调方法**：保留冻结的视觉编码器和 LLM，仅微调投影器 → 每个前向传播中，投影器输出经过相同的顶部主成分去除 → 使用标准语言建模损失优化 → 微调后模型在推理时可直接使用（无需再次去偏）。

## 6. 主要结论与发现
1. **过度对齐导致语义掩蔽**：视觉嵌入在早期至中层解码器层被强迫对齐到文本流形，顶部主成分充当“模态锚点”，掩盖了真实视觉特征。
2. **语言偏差是通用的**：不同数据集（COCO、ChartQA、DocVQA 等）的文本流形顶部主成分高度相似，表明这是一种与数据集无关的统计偏差。
3. **训练‑免去偏显著减少幻觉**：在 Qwen2.5-VL-7B 上，AMBER CHAIR 从 5.14 降至 4.23（↓17.7%），HAL 从 28.62 降至 25.07（↓12.4%）；在 LLaVA-1.5-7B 上，CHAIR 从 9.20 降至 6.74（↓26.7%）。同时 POPE 等能力指标保持或微升，未发生退化。
4. **微调方法进一步提升长描述质量**：在 VRSBench 上，CLAIR 分数从标准 SFT 的 72.30 提升至 74.70；在 TextCaps 上从 60.30 提升至 63.80（LLaVA-1.5-7B）。
5. **Logit Lens 分析证实**：顶部主成分移除后，视觉 patch 的解码语义从标点、冠词等结构词转变为具体物体（如“bench”“refrigerator”），表明视觉特征被重新激活。
